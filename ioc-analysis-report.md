# Threat Intelligence & IOC Analysis Report

## Summary
This lab demonstrates the investigation of Indicators of Compromise (IOCs) including IP addresses, domains, and file hashes using threat intelligence platforms.

## Tools Used
- AbuseIPDB
- VirusTotal

## Investigation Process
1. Identified suspicious IP address
2. Reviewed abuse reports and reputation score
3. Cross-referenced findings using VirusTotal
4. Investigated domain and file hash indicators

## Key Findings
- The IP address showed a high abuse confidence score
- Multiple reports indicated brute-force and scanning activity
- VirusTotal confirmed malicious classification by multiple vendors
- Domain and file hash analysis supported threat classification

## Conclusion
The analyzed indicators were confirmed to be associated with malicious activity, demonstrating the ability to use threat intelligence tools for IOC validation and investigation.
