# 🌐 Threat Intelligence & IOC Analysis Lab

## Overview
This project demonstrates the investigation and analysis of Indicators of Compromise (IOCs) including IP addresses, domains, and file hashes using threat intelligence platforms.

## Tools Used
- AbuseIPDB
- VirusTotal

## Skills Demonstrated
- Threat intelligence analysis
- IOC investigation
- Cross-referencing security data
- Malicious activity identification

## Project Structure
- screenshots/ → Evidence of analysis
- reports/ → IOC investigation report

## Investigation Highlights
- Identified high-risk IP addresses
- Validated malicious indicators across multiple platforms
- Analyzed domains and file hashes for threat classification

## Outcome
Successfully identified and validated malicious indicators, demonstrating real-world threat intelligence and analysis skills.
